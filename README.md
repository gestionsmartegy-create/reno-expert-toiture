# RenoExpert Toiture — Site Web

Site HTML complet pour une entreprise de toiture au Québec.

## 🌐 Voir le site en ligne
👉 [https://gestionsmartegy-create.github.io/reno-expert-toiture/reno_expert_toiture.html](https://gestionsmartegy-create.github.io/reno-expert-toiture/reno_expert_toiture.html)

## 📁 Fichier principal
- `reno_expert_toiture.html` — site complet (single-page, aucune dépendance externe)

## 📋 Pages incluses
- Accueil
- Installation de toiture
- Remplacement de toiture
- Bardeaux d'asphalte
- Estimation gratuite (formulaire + photos + adresse)
- Contact

## 🔄 Reprendre le travail sur un autre ordi

```bash
git clone https://github.com/gestionsmartegy-create/reno-expert-toiture
cd reno-expert-toiture
```

Puis ouvre `reno_expert_toiture.html` dans ton navigateur.

```bash
git add reno_expert_toiture.html
```
